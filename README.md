# 📚 Quiz Gestión de Proyectos

Una aplicación web interactiva para evaluar conocimientos sobre gestión de proyectos con 51 preguntas de opción múltiple.

## Características

✅ **10 preguntas al azar** de un banco de 51 preguntas  
✅ **Una pregunta por vez** con navegación clara  
✅ **Validación inmediata** - sabe si acertaste o fallaste  
✅ **Justificaciones detalladas** para cada respuesta correcta  
✅ **Imágenes incluidas** en preguntas sobre diagramas (preguntas 7, 19, 20)  
✅ **Resumen de errores** al finalizar con explicaciones  
✅ **Interfaz moderna y responsiva**  
✅ **Funciona offline** - no requiere internet después de cargar  

## Temas Cubiertos

- 🛠️ Herramientas de Gestión (Trello, Notion, Miro, Chat GPT)
- 👥 Roles y Fases del Proyecto
- 📊 Metodologías (Scrum, Kanban, Waterfall, Espiral)
- 📈 Planificación y Alcance
- 🔗 Diagramas de Red y CPM
- 📅 Diagramas de Gantt
- ⚡ Metodologías Ágiles vs Tradicionales

## Cómo Usar

1. Abre `index.html` en tu navegador
2. El quiz selecciona 10 preguntas al azar
3. Lee la pregunta y selecciona tu respuesta
4. Recibirás retroalimentación inmediata
5. Al finalizar, verás el resumen de errores con justificaciones
6. Haz clic en "Volver a Comenzar" para jugar nuevamente
